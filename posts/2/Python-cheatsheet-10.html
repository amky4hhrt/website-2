Here's the simple cheat sheet for python:


1 Checking Python Version: 

Windows: In CMD, type python -version

Linux: In terminal, type python3


2 Print Function:

ex: print("Hello World")

>>Hello World


3 Variables - Storing Data: 

ex: message = 'I am a good programmer'

     print(message)

>>I am a good programmer


4 String - Data Type - Series of Characters

ex: messenger = 'I am a messenger'

     print(type(messenger)

>><class 'str'>


5 Changing case in String 

i: Using title() method 

ex: name = 'ama and lovely'

     print(name.title())

>>Ama And Lovely


ii: Using upper() method

ex: name = 'ama and lovely'

     print(name.upper())

>>AMA AND LOVELY


iii: Using lower() method

ex: name = 'AMA AND LOVELY'

      print(name.lower())

>>ama and lovely


6 Concatenating Strings: 

ex: first_name = 'Ama'

     last_name = 'Lovely'

     full_name = first_name + ' ' + 'last_name

     print(full_name)

>>Ama Lovely



7  Stripping Whitespaces:

i: Strip from right - rstrip() method

ex: name = 'The hard Tortoise '

     name = name.rstrip()

     print(name)

>>The hard Tortoise


ii Strip from left - lstrip() method

ex: name = '   Ama'

      name = name.lstrip()

      print(name)

>>Ama


iii Strip from both sides - strip() method

ex: name = '   Ama  Lovely  '

      name = name.strip()

      print(name)

>>Ama Lovely


8 Avoid Syntax errors in Strings

ex: message = 'There's so many programming languages, but python is my favourite'

     print(message)

>> -- error -- 'There's , python cannot process three single quotes within one sentence.


ex: messsage = "There's so many programming languages, but python is my favourite"

      print(message)

>>There's so many programming languages, but python is my favourite.


ex: Using /' and /"

i: 

message = 'There/'s so many programming languages, but python is my favourite'

print(message)

>>There's so many programming languages, but python is my favourite.


ii:

message = "There/'s so many programming languages, but python is my favourite"

print(message)

>>There's so many programming languages, but python is my favourite.


9 Integers - numberical values - can perform calculation

ex: num1 = 5

      num2 =  10

      print(num1 + num2)

>>15

     print(num2 - num1)

>>5

     print(num1 * num2)

>>150

     print(num2/num1)

>>2

 

10 Type error - convert number to string - str() function

ex: age = 24

      message = 'Happy ' + age + 'th birthday!'

      print(message)

>> --error -- TypeError


ex: age = 24 

     message = 'Happy ' + str(age) + 'th Birthday!'

     print(message)

>>Happy 24th Birthday!
