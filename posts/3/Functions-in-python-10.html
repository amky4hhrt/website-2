Functions in Python: 10 - part 1

-- Functions are the named blocks of code which can be called to perform specific task.


1 print()

-- The print functions prints the content to the screen, which is shown as output.


ex: print('I am a good programmer')
    
>>I am a good programmer


2 str()

--The string function converts the numerical values to the string data type.

ex: my_num = 12
      my_num = str(my_num)
      print(my_num)

     print(type(my_num))

>><class 'str'>


3 title()

-- The title function converts the string to the title case.

ex: message = 'i am a good programmer'
      print(message.title())

>>I Am A Good Programmer.


4 rstrip(), lstrip(), strip()

--These functions strip the whitespaces from right side, left side and both respectively.

i rstrip()

ex: message = 'Amy Lovely  '
      print(message.rstrip())

>>Amy Lovely

ii lstrip()

ex: message = '   I am a good programmer.'
      print(message.lstrip())

>>I am a good programmer.

iii strip()

ex: message = ' I am a good programmer.  '
      print(message.strip())

>>I am a good programmer.


5 sorted()

--It sorts the input data by ascending order by default.

ex: my_data = ['Tiger', 'Lion', 'Giraffee', 'Elephant', 'Deer', 'Rhino']
      my_data = sorted(my_data)
      print(my_data)

>>['Deer','Elephant','Giraffee', 'Lion', 'Rhino', 'Tiger']


6 len()

-- This function outputs the length of the data

ex: my_data = ['Cat', 'Dog', 'Rabbit', 'Parrot']
      print(len(my_data))

>>4


7 range()

-- The range function is used to output a range of numbers.

ex: for value in range(1, 5):
          print(value)
 
>>1
>>2
>>3
>>4


8 list()

-- This function is used to create a list.

ex: numbers = list(range(1,5))
      print(numbers)

>>[1,2,3,4]


9 sum()

--The sum function is used to sum all the input values.

ex: my_numbers = [1,2,3,4,5]
      print(sum(my_numbers))

>>15


10 lower(), upper()

--These functions are used to make the string a lowercase or an uppercase respectively.

i lower()

ex: message = 'The skill number 1, Daily programming.'
      print(lower(message))

>>the skill number 1, daily programming.

ii upper()

ex: message = 'The skill number 1, Daily programming.'
      print(upper(message))

>>THE SKILL NUBER 1, DAILY PROGRAMMING.
