Functions in Python 8 - Part 2


11 type()

-- The type function outputs the data type of the input data.

ex: message = 'I am a programmer'

      print(type(message))

>><class 'str'>


12 input()

-- The input function takes input from the user.

ex: message = 'What is your age?'

      age = input('Enter your age:')

      print(message)

      print('Your age is ' + str(age) + '.')

>>What is your age?

<<(input - age by user)

>>Your age is 30.


13 int()

-- This function converts the float data to integer data by removing decimal.

ex: my_float = 33.3333

      print(int(my_float))

>>33


14 pop()

-- This function will remove the last data and output it one time.

i

ex: a_list = [1,2,3,4,5,6,7]

      a_list.pop()

>>7

ii

ex: a_list = [1,2,3,4,5,6,7]

      b_list = a_list.pop()

      print(a_list)

      print(b_list)

>>[1,2,3,4,5,6]

>>7


15 remove()

-- This function is used to remove any value inside a list.

ex: zoo_list = ['Tiger','Zebra','Deer','Lion','Rhino']

      zoo.list.remove('Zebra')

      print(zoo_list)

>>['Tiger','Deer','Lion','Rhino']


16 randint()

-- This function will output a random integer from the given range.

ex: from random import randint

      x = randint(1,10)

      print(x)

>>4


17 max(), min()

-- These functions will output the maximum or minimum value inside a list respectively.

i max()

ex: a_list = [1,2,3,4,5,6,7]

      print(max(a_list))

>>7

i min()

ex: a_list = [1,2,3,4,5,6,7]

      print(min(a_list))

>>1


18 float()

-- This function converts an integer value to the float value.

ex: my_num = 5

      print(float(my_num))

>>5.0
