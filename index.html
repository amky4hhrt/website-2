Python cheat sheet - part 1

July 14, 2024 by Ankit

1 Checking Python Version: 

Windows: In CMD, type python -version

Linux: In terminal, type python3


2 Print Function:

ex: print("Hello World")

>>Hello World


3 Variables - Storing Data: 

ex: message = 'I am a good programmer'

     print(message)

>>I am a good programmer


4 String - Data Type - Series of Characters

ex: messenger = 'I am a messenger'

     print(type(messenger)

>><class 'str'>


5 Changing case in String 

i: Using title() method 

ex: name = 'ama and lovely'

     print(name.title())

>>Ama And Lovely


ii: Using upper() method

ex: name = 'ama and lovely'

     print(name.upper())

>>AMA AND LOVELY


iii: Using lower() method

ex: name = 'AMA AND LOVELY'

      print(name.lower())

>>ama and lovely
